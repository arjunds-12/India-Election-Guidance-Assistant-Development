# Election Assistant

A premium, interactive web application to help users understand the election process, timelines, and steps.

## Features
- **Interactive Timeline**: Visual guide to election milestones.
- **Voter Roadmap**: Step-by-step guidance (Register, Prepare, Vote).
- **Smart Assistant**: Keyword-based chat interface for common queries.
- **Modern UI**: Glassmorphic design with Outfit typography.

## Tech Stack
- HTML5 / Vanilla JavaScript
- CSS3 (Custom Design System)
- Vite (Dev Server & Build Tool)

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Recommended: LTS)

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Deployment
To build for production:
```bash
npm run build
```
The output will be in the `dist` folder.
